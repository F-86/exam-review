# exam-review

📚 大学生期末复习 Skill — 基于 **Harness 6 概念**设计的 AI 复习工具。

## 概述

给用户**自己用**的复习工具。你提供教材/笔记 + 学习天数，它自动生成：

- **考点清单** — 按天拆分的结构化知识点，含定义、公式、常见题型、易混淆概念
- **自测题** — 每天配套的练习（选择 + 简答 + 计算），含详细答案解析

## 工作流程

```
你给材料 + 说天数
        ↓
我出 Day 1 样板 → 你审格式、内容
        ↓ (确认 OK)
我批量出剩余天数 → 你自用，不打扰
        ↓ (你什么时候要我什么时候做)
你要总结/给错题 → 我生成针对性材料
```

## 用法

加载 skill 后，直接发来你的教材/笔记 + 说几天学完即可。

## 文件结构

```
.review/
├── plan.md                           # 复习计划
├── 考点清单-{科目}-Day{1..N}.md       # 每天一份考点
├── 自测题-{科目}-Day{1..N}-第1轮.md   # 每天一份自测题（含答案）
└── 总结-{日期}.md                     # 用户要求时才生成
```

## 配套 Hermes 配置

```bash
hermes skills link /path/to/exam-review
# 或复制到 ~/.hermes/skills/productivity/exam-review/
```

## Harness 6 概念

| 概念 | 本 Skill 的实现 |
|------|----------------|
| 上下文管理 | 渐进加载：只看当前阶段需要的参考文件 |
| 工具系统 | 文件读取 / OCR；输出 Markdown 文档 |
| 执行编排 | 4 Step 线性流程：收集→样板→批量→自用 |
| 状态与记忆 | `.review/` 工作区文件 |
| 评估与观测 | 自测题含答案 + 解析，用户自行评估 |
| 约束与恢复 | 不做解题器，用户确认后才批量生成 |
